# New Hotel Mahadev Website

A premium web application for New Hotel Mahadev, located in the heart of the Kedarnath pilgrimage route. This site provides information about rooms, amenities, restaurant services, and easy booking options for pilgrims and tourists.

## Features

- **Premium Room Showcase**: Detailed views of various room types with high-quality gallery images.
- **In-House Restaurant**: Explore delicious vegetarian meal options available at the hotel.
- **Mobile Optimized**: Seamless booking and browsing experience across all devices.
- **Real-time Connectivity**: Integrated WhatsApp support for instant inquiries and bookings.
- **Modern Infrastructure**: Built with high-performance technologies for fast loading times.

## Tech Stack

- **Frontend**: React.js with TypeScript
- **Styling**: Tailwind CSS & shadcn/ui
- **Build Tool**: Vite
- **Backend / Database**: Supabase
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dmash10/hotel-mahadev.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main application pages and routes.
- `src/hooks`: Custom React hooks for data fetching and state management.
- `src/lib`: Configuration for external services like Supabase.

## Deployment

This project is ready to be deployed on platforms like **Vercel** or **Netlify**. For production builds, run:

```bash
npm run build
```
